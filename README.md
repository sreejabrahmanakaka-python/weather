# weatherWeather_App
🌦️ Weather App – Python Tkinter Project
📌 Project Overview
This is a simple Weather Application built using Python Tkinter. It takes a city name as input and fetches real-time weather data using the OpenWeatherMap API.
🚀 Features
Enter city name to get weather details
Displays temperature in Celsius
Shows feels-like temperature
Displays humidity and wind speed
Shows weather condition (clear, cloudy, etc.)
Error handling for invalid city names
Handles network issues properly
🛠️ Technologies Used
Python
Tkinter (GUI)
Requests library
OpenWeatherMap API
🌐 API Used
OpenWeatherMap API
https://openweathermap.org/
▶️ How to Run
Install required library: pip install requests Add your API key in the code: API_KEY = "your_api_key_here" Run the program: python weather_app.py
📊 Output
After entering a city name, the app displays:

City Name
Temperature (°C)
Feels Like Temperature
Humidity (%)
Wind Speed (m/s)
Weather Condition
⚠️ Error Handling
Shows error if city name is empty
Shows error if city is not found
Shows error for network issues
